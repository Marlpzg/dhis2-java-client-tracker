# dhis2-java-client

DHIS 2 API client for Java and Spring.

### Getting started

To use `dhis2-java-client` with Maven you can specify the following dependency:

```xml
<dependency>
  <groupId>org.hisp</groupId>
  <artifactId>dhis2-java-client</artifactId>
  <version>1.0.0</version>
</dependency>
```
